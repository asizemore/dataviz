Multilayer networks are used to represent many types of systems, such as those that have multiple types of connections between nodes. Above we see a multilayer network with five layers (left) and its corresponding supra-adjacency matrix (right) which records the presence of connections between nodes.

  • Hover over a node in the network to see it's corresponding location and connections in the supra-adjacency matrix.

  • Choose to color nodes by layer or by node identity.

  • Rotate the multilayer network 90 degrees.

  • Adjust the viewing angle with the sliders.
